DVWA Firewall Security Lab
Overview

This project demonstrates how a vulnerable web application can be exploited through SQL Injection and how network-layer security controls can be used to reduce exposure.

A local security lab was built using Kali Linux, Ubuntu Server, DVWA, Apache2, and UFW Firewall. The objective was to understand the difference between application-layer vulnerabilities and network-layer security controls.

This lab was created for educational purposes in an isolated environment.

Objectives

The objectives of this project were:

Understand how SQL Injection vulnerabilities work
Explore the impact of insecure user input handling
Configure and test firewall rules using UFW
Verify security controls using Nmap and Curl
Learn the difference between application security and network security
Practice building and documenting a cybersecurity lab  


Lab Environment
Component	Technology
